## Doctor User Stories
- As a Doctor, I want to view my appointments by date so that I can manage my schedule.
- As a Doctor, I want to see patient details for each appointment so that I can prepare for consultations.
- As a Doctor, I want to view my booked appointments so that I can track my patients.

## Patient User Stories
- As a Patient, I want to register and log in so that I can access the system.
- As a Patient, I want to search doctors by name or specialty so that I can find the right doctor.
- As a Patient, I want to book an appointment at an available time so that I can receive medical care.
- As a Patient, I want to view my appointments so that I can track my bookings.

## Admin User Stories
- As an Admin, I want to add doctors to the system so that they are available for appointments.
- As an Admin, I want to manage doctors and patients so that system data remains accurate.
